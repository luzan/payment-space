# Week 5-6: Resources & Reading Materials

## Card Network Documentation

### Visa

- **Visa Core Rules - Dispute Management**
  - Focus on: dispute timeframes, reason codes, evidence requirements

- **Visa Dispute Monitoring Program Guide**
  - Focus on: thresholds, remediation requirements, fine schedules

- **Visa Fraud Monitoring Program Guide**
  - Focus on: fraud ratio calculations, program tiers

### Mastercard

- **Mastercard Chargeback Guide**
  - Focus on: reason codes, compelling evidence requirements

- **Mastercard Excessive Chargeback Program**
  - Focus on: ECP thresholds, ECM requirements

- **Mastercard Excessive Fraud Merchant Program**
  - Focus on: fraud monitoring requirements

## PCI Security Standards

### PCI Security Standards Council

- **Website:** pcisecuritystandards.org
- **PCI DSS v4.0 Standard**
  - The full standard document
  - Focus on: all 12 requirements, Appendix A (service providers)

- **PCI DSS Quick Reference Guide**
  - Simplified overview of requirements

- **SAQ (Self-Assessment Questionnaire) Documents**
  - Study SAQ-D for service providers
  - Understand which controls apply

### PCI Guidance Documents

- **Tokenization Guidelines**
  - Focus on: scope reduction, implementation requirements

- **P2PE (Point-to-Point Encryption) Standard**
  - Focus on: encryption requirements, key management

- **Information Supplement: Multi-Factor Authentication**
  - Focus on: MFA requirements for CDE access

## AML/BSA Resources

### FinCEN

- **Website:** fincen.gov
- **BSA/AML Examination Manual**
  - Focus on: MSB (Money Services Business) requirements

- **SAR Filing Instructions**
  - Focus on: filing thresholds, narrative requirements

- **AML Program Requirements**
  - Focus on: five pillars of AML compliance

### FFIEC (Federal Financial Institutions Examination Council)

- **BSA/AML Examination Manual**
  - Comprehensive guide to AML compliance expectations
  - Focus on: transaction monitoring, suspicious activity detection
